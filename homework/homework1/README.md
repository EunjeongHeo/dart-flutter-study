## [필수문제1] 점수에 맞는 등급 출력 프로그램
#### 문제 설명
주어진 점수에 대해 조건문을 활용하여 해당 점수에 알맞은 등급(A/B/C)을 판정하고 출력하는 프로그램이다.

####  조건
- 점수: 84점
- 등급 기준:
  - 90점 이상 ~ 100점 미만 → A등급
  - 80점 이상 ~ 90점 미만 → B등급
  - 80점 미만 → C등급

####  출력 형태
```
이 학생의 점수는 [점수]점 이며, 등급은 [등급]등급입니다!
```


## [필수문제2] 가상의 쇼핑몰 결제 프로그램
#### 문제 설명
장바구니 목록과 상품별 가격을 기반으로 총 금액 계산, 할인 여부 판단, 최종 결제 금액 출력 기능을 구현하는 프로그램이다.

#### 조건
- 상품 가격표
  - 티셔츠: 10000원
  - 바지: 8000원
  - 모자: 4000원

- 장바구니 예시
  - ['티셔츠', '바지', '모자', '티셔츠', '바지']

- 할인 조건
  - 총 금액이 20,000원 이상이면 10% 할인 적용

#### 출력 형태 예시
```
장바구니에 40000원 어치를 담으셨네요 !
할인 금액 : 4000원
최종 결제 금액은 36000원입니다!
```

## [도전문제] 가상의 복권(로또) 프로그램
#### 문제 설명
1~45 사이의 숫자 중 **중복되지 않게 6개의 번호를 추첨하여 로또 번호를 발급**한 뒤,
지정된 당첨 번호와 비교하여 **등수를 판정**하고, 마지막으로 로또 번호를 **초기화하는 프로그램**이다.

#### 조건
- 로또 번호는 1~45 범위의 숫자 중 중복 없이 6개 생성
- 당첨 번호: `[9, 19, 29, 35, 37, 38]`
- 일치한 번호 개수에 따른 등급
  - 5개 이상: **1등**
  - 4개: **2등**
  - 3개: **3등**
  - 2개 이하: **당첨 실패**
- 마지막 단계에서 로또 번호를 `[]`로 초기화하여 출력

#### 출력 형태 예시
```
발급한 로또 번호  : [22, 37, 24, 6, 23, 9]
당첨 여부 : 당첨 실패
현재 발급한 로또 번호: []
```
