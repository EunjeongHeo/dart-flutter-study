## [필수문제1] 점수에 맞는 등급 출력 프로그램
#### 문제 설명
주어진 점수에 대해 조건문을 활용하여 해당 점수에 알맞은 등급(A/B/C)을 판정하고 출력하는 프로그램이다.

####  조건
- 점수: 84점
- 등급 기준:
  - 90점 이상 ~ 100점 미만 → A등급
  - 80점 이상 ~ 90점 미만 → B등급
  - 80점 미만 → C등급

####  출력 형태
```
이 학생의 점수는 [점수]점 이며, 등급은 [등급]등급입니다!
```
