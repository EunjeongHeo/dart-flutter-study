## [필수문제1] 점수에 맞는 등급 출력 프로그램
#### 문제 설명
주어진 점수에 대해 조건문을 활용하여 해당 점수에 알맞은 등급(A/B/C)을 판정하고 출력하는 프로그램이다.

####  조건
- 점수: 84점
- 등급 기준:
  - 90점 이상 ~ 100점 미만 → A등급
  - 80점 이상 ~ 90점 미만 → B등급
  - 80점 미만 → C등급

####  출력 형태
```
이 학생의 점수는 [점수]점 이며, 등급은 [등급]등급입니다!
```


## [필수문제2] 가상의 쇼핑몰 결제 프로그램
#### 문제 설명
장바구니 목록과 상품별 가격을 기반으로 총 금액 계산, 할인 여부 판단, 최종 결제 금액 출력 기능을 구현하는 프로그램이다.

#### 조건
- 상품 가격표
  - 티셔츠: 10000원
  - 바지: 8000원
  - 모자: 4000원

- 장바구니 예시
  - ['티셔츠', '바지', '모자', '티셔츠', '바지']

- 할인 조건
  - 총 금액이 20,000원 이상이면 10% 할인 적용

#### 출력 형태 예시
```
장바구니에 40000원 어치를 담으셨네요 !
할인 금액 : 4000원
최종 결제 금액은 36000원입니다!
```
