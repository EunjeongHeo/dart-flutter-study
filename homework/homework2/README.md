## [필수문제] 학생의 통계 확인하기
#### 문제 설명
파일(students.txt)에 저장된 학생 점수 데이터를 불러온 뒤, 사용자로부터 학생 이름을 입력 받아, 해당 학생의 점수를 콘솔에 출력하고 결과를 파일로 저장하는 프로그램이다.

####  흐름
1. 프로그램 시작 시 student.txt 학생 데이터 파일을 읽어옵니다.
2. 사용자에게 조회할 학생 이름을 입력 받습니다.
3. 입력한 이름이 존재하지 않으면 다시 입력을 요청합니다.
4. 올바른 이름이 입력되면 학생 점수를 출력합니다.
5. 결과를 result.txt 파일로 저장한 뒤 프로그램을 종료합니다.

####  입력 형태 (students.txt)
```
홍길동,90
김철수,80
```

####  출력 형태 (콘솔)
```
어떤 학생의 통계를 확인하시겠습니까?
김영희
잘못된 학생 이름을 입력하셨습니다. 다시 입력해주세요.
어떤 학생의 통계를 확인하시겠습니까?
홍길동
이름: 홍길동, 점수: 90
저장이 완료되었습니다.
```

#### 결과 형태 (result.txt)
```
이름: 김철수, 점수: 80
```
